Toys api documentation



Routes
All products:Get /toys
Producs by category: /cat/:catId
Add product: use the link in postman and use POSTadd
Edit product: use the link in postman and use PUT /edit
Delete product: use the link in postman and use DEL+toys id/delete
To sort minimum to maximum prices for example:20 to 45 use /?min=20&max=45/prices
